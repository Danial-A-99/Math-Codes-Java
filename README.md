# Math-Codes-Java
This repository contains different programs that can be used for certain operations. All code is in Java, and it encompasses the field of math. Enjoy Using Them!   

These Programs Are All from my School CompSci class through homework and assignments.

This Repository Includes:
- Ambiguous Case Determiner
- Asymptote Finder
- Cosine Law Calculator
- Heron's Formula Program
- Quadratic Formula Calculator
- Newton's Method Program
- Power Rule Program
- PI Approximations Program
- Polynomial Evaluator
- Sine Law Calculator
